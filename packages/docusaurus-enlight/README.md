# `@openbayes/docusaurus-enlight`

Docusaurus Enlight theme made by OpenBayes

## Install

```bash
yarn add @openbayes/docusaurus-enlight
```

## Usage

In React:

```jsx
import '@openbayes/docusaurus-enlight';
```

In Sass with Webpack:

```scss
@import "~@openbayes/docusaurus-enlight";
```
