# `@openbayes/docsearch-css`

Algolia DocSearch CSS powered by Kladenets color system for OpenBayes

## Install

```bash
yarn add @openbayes/docsearch-css kladenets@next
```

## Usage

In React:

```jsx
import 'kladenets';
import '@openbayes/docsearch-css';
```

In Sass with Webpack:

```scss
@import '~kladenets';
@import "~@openbayes/docsearch-css";
```
